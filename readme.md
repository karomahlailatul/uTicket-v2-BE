## uTicket-Back-End

Instal Node.js on here : https://nodejs.org/en/download/
Instal PostgreSQL on here : https://www.postgresql.org/download/

Install third party npm modules

```shell
npm i ;
```

## Testing

```shell
npm run dev;
```

```shell
npm run server;
```

## Debugging

```shell
npm run lint;
```
